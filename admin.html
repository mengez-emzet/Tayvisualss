Quick Guide

Available — date is open for bookings. Clients can select it.

Booked — date is taken. Add a session label so you remember what's on.

Unavailable — date is blocked (holiday, personal, travel). Clients see it struck through.

Changes are live instantly — clients see updates on their calendar without refreshing.
